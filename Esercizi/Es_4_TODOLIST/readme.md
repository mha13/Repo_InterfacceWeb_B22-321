# TodoList
Crea una todolist per gli utenti

Costruisci i seguenti campi in HTML: input testuale, data, pulsante aggiungi.
Ogni volta che l'utente digita nei campi e modifica la data cliccando sul pulsante viene creato un oggetto "Item" (singolo todo).
